Text format for spectrum licence site data upload

The page provides the required text format and validation rules for uploading spectrum licence site data in CSV files to ISED’s Spectrum Management System, including field descriptions, examples, and error conditions for compliance. [ised-isde.canada.ca]
A
Spectrum licence number

Indicates the number of your Spectrum licence.

Example
010898101

Validation rules

  • The licence number must be a number in range 1 to 999999999.
  • Do not include the licence version number.
  • Must be a granted (paid) and not expired spectrum licence.
  • Must belong to an account in your web profile (external uploads only).

Validation errors

If the licence number does not contain a leading 0, does not belong to your account, the licence number is not in Granted status, or the licence is expired.
B
Upload reference number

Optional field which could be used to store a reference name for the dataset.

Example
Monthly upload, March 2023

Validation rules

  • Optional field.
  • Maximum length 63 characters.
C
Contact name

Contact name of the person who should be contacted regarding this data.

Example
John Smith

Validation rules

  • Mandatory field.
  • Maximum length 254 characters.

Validation errors

The Contact Name is missing or exceeds the character limit.
